# Преобразование систем координат
## План выполнения работы:
1. Создать новый проект WPF Application в среде разработки Visual Studio.
2. Добавить на форму Canvas как главный контейнер для размещения элементов управления.
3. Разместить в Canvas элементы управления различных типов (например, Button, TextBox, Ellipse).
4. Реализовать код, который определяет локальные координаты элемента управления (например, центр Ellipse) и преобразует их в координаты всего приложения (в координаты окна).
5. Реализовать код, который принимает точку на уровне приложения и преобразует её в локальные координаты элемента управления.
6. Добавить кнопку, которая при нажатии будет отображать текущие локальные и глобальные координаты выбранного элемента управления.
.Убедиться, что код корректно учитывает трансформации, примененные к элементам управления (если таковые имеются).
